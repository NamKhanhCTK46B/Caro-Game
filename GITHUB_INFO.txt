MÔ TẢ DỰ ÁN
-----------
Trò chơi Cờ Caro (Gomoku) với giao diện JavaFX và 3 mức độ AI khác nhau.
Game desktop cross-platform (Windows/macOS/Linux) với giao diện hiện đại
và thuật toán AI từ Easy đến Hard (Minimax + Alpha-Beta Pruning).

TÍNH NĂNG CHÍNH
---------------
- Bàn chơi 15x15 (Gomoku/Caro chuẩn)
- 3 mức độ AI:
  * Easy: Random AI (chơi ngẫu nhiên)
  * Medium: Heuristic AI (đánh giá pattern)
  * Hard: Minimax AI với Alpha-Beta Pruning
- Phát hiện tự động 5 quân liên tiếp (ngang/dọc/chéo)
- Highlight dòng thắng
- Undo/Redo không giới hạn
- Theo dõi điểm số liên tục
- Giao diện JavaFX mượt mà với CSS styling

CÔNG NGHỆ SỬ DỤNG
-----------------
Trạng thái:      Đang phát triển
- Design Patterns: MVC, Strategy, Observer, Memento, Singleton
- Version Control: Git

KỸ THUẬT PHẦN MỀM
-----------------
Trạng thái:      Đang phát triển
  - Observer Pattern: Tự động cập nhật UI
  - Memento Pattern: Lưu/phục hồi trạng thái game
  - Singleton Pattern: Quản lý điểm số tập trung

AI Algorithms:
  - Easy: Random move selection
  - Medium: Heuristic evaluation (pattern matching)
  - Hard: Minimax with Alpha-Beta Pruning (depth 3-4)

HƯỚNG DẪN BUILD
---------------
1. Clone repository:
   git clone https://github.com/NamKhanhCTK46B/Caro-Game.git
   cd Caro-Game

2. Build project:
   mvn clean package

3. Chạy ứng dụng:
   java -jar target/caro-game.jar

4. Hoặc sử dụng scripts:
   .\scripts\build.bat       (Build project)
   .\scripts\run.bat         (Chạy ứng dụng)
   .\scripts\optimize.bat    (Tối ưu dung lượng)
